# 🚀 Quick Start Guide

## For Web App (PWA)

Just open `index.html` in your browser! That's it!

## For Native Mobile App

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Add Platforms
```bash
# For Android
npx cap add android

# For iOS (Mac only)
npx cap add ios
```

### Step 3: Sync and Open
```bash
# Android
npm run android

# iOS (Mac only)
npm run ios
```

### Step 4: Build and Test
- Android Studio or Xcode will open
- Select an emulator/device
- Click Run ▶️

## What You Get

✅ **Web App** - Works in any browser, installable as PWA
✅ **Android App** - Native Android app, ready for Play Store
✅ **iOS App** - Native iOS app, ready for App Store
